# results/

Generated figures, tables, and frontend-readable experiment runs land here
(e.g. `runs/<run_id>/manifest.json`, `patches.jsonl`, `readouts.jsonl`).

Large binaries (cached activations, fitted `J` tensors) are gitignored;
commit only small, human-readable artifacts.

`runs/demo-qwen25-7b-l18/` is synthetic demo data generated by
`experiments/build_demo_run.py` for frontend development. It is not an
experimental result and should not be cited as evidence.
